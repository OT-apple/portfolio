# Portfolio images

Place your photos in this folder:

| File | Where it appears |
|------|------------------|
| `profile.jpg` | Sidebar avatar |
| `about.jpg` | Portfolio / about cards |
| `education.jpg` | Education visuals |
| `internship.jpg` | Robotics projects |
| `experience.jpg` | Experience / SharePoint |
| `project1.jpg` | Portfolio projects |
| `project2.jpg` | Portfolio projects |
| `certificates.jpg` | Certificates section |
| `contact.jpg` | Optional contact visual |

Missing images show a gradient placeholder automatically.
